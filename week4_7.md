### Pandoc

Rename it as "2html"

open [2html](code/2html) in gedit

```

pandoc -s -o week1.html week1.md

pandoc -s -o week2.html week2.md

pandoc -s -o week3.html week3.md

pandoc -s -o week4.html week4.md

git pull

```
